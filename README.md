# Soroban Exam

Get a soroban exam from www.sorobanexam.org.

## Install requirements

```
pip install -r requirements.txt
```

## Useage

```bash
./soroban.py --level 6
```

```bash
usage: soroban.py [-h] [-l LEVEL]

options:
  -h, --help            show this help message and exit
  -l LEVEL, --level LEVEL
                        level of exam (easiest 9 - hardest 1)
```
